## 4. Body (1:00–4:30)

> "Lens 6 set up phase-aware protocol. Today: *compare phase-locked methodology with simpler averaged-only methodology*. Where do they differ?
>
> *[pause]*
>
> *Averaged-only methodology.*
>
> *Acquire coil signal continuously*; *compute statistical average over many cycles (or just RMS value)*; *report single number*. *Mainstream simple measurement.*
>
> *Output*: *one value per measurement* (e.g., 'coil RMS voltage at f_res = 35 mV').
>
> *Information loss*: *time-domain pattern within each cycle is averaged out*; *only aggregate behaviour preserved*.
>
> *Activation-phase (phase-locked) methodology.*
>
> *Acquire coil signal phase-locked to drive*; *separate samples by phase*; *report phase-amplitude pattern (6 values per cycle, averaged across many cycles)*. *Wk 31 Mod C content.*
>
> *Output*: *6 values per measurement* (e.g., 'phase 0 = 28 mV; phase 1 = 35 mV; phase 2 = 42 mV; phase 3 = 28 mV; phase 4 = 35 mV; phase 5 = 42 mV').
>
> *Information preservation*: *time-domain pattern within cycle preserved*; *phase-amplitude variation visible*.
>
> *[pause]*
>
> *Three structural similarities.*
>
> *Similarity one — both use mainstream lab equipment.* *Signal generators, oscilloscopes, computer-side processing*; *both use standard equipment*.
>
> *Similarity two — both characterise coils.* *Both produce engineering data about coil behaviour*. *Both legitimate engineering practice.*
>
> *Similarity three — both replicable.* *Standard methodologies; replicate-able by other practitioners*.
>
> *Three structural differences.*
>
> *Difference one — information content.* *Averaged-only: 1 value per measurement. Phase-locked: 6 values per measurement.* *Phase-locked is 6× more informative within cycle.*
>
> *Difference two — equipment requirements.* *Averaged-only: basic oscilloscope or AC voltmeter. Phase-locked: oscilloscope with external trigger; signal generator with phase-trigger output.* *Phase-locked needs more capable equipment.*
>
> *Difference three — analysis complexity.* *Averaged-only: read display value. Phase-locked: phase-binning, averaging, comparison; computer-side processing.* *Phase-locked is more analytically demanding.*
>
> *[pause]*
>
> *Three engineering trade-offs.*
>
> *Trade-off 1 — information vs simplicity.* *Phase-locked gives more information; requires more setup. Averaged-only is simpler; less information.*
>
> *Trade-off 2 — equipment cost.* *Phase-locked rig is more expensive; averaged-only cheaper.*
>
> *Trade-off 3 — analysis time.* *Phase-locked needs computer processing; averaged-only is direct read.*
>
> *Three trade-offs; engineering choice depends on testing goal*.
>
> *[pause]*
>
> *When each methodology is appropriate.*
>
> *Averaged-only*: *when only aggregate coil behaviour matters* (e.g., 'is the coil resonating? what's its Q-factor?'). *Standard inductor characterisation.*
>
> *Phase-locked*: *when within-cycle pattern matters* (e.g., 'is there a 6-step pattern? does sequence match doubling-cycle?'). *Aetheron-distinctive testing requires phase-locked.*
>
> *VBM-tradition's specific predictions* (Wk 30 Mod B Lens 6) *require phase-locked methodology to test*; *averaged-only methodology cannot detect within-cycle pattern; cannot test Aetheron-distinctive predictions*.
>
> *[pause]*

>
> *Observation one — phase-locked is necessary for VBM-distinctive testing.* *Without phase-locking, predictions can't be empirically engaged at full*. *Methodology shapes what's testable.*
>
> *Observation two — averaged-only complements as baseline.* *Averaged-only measurements give Q-factor, resonance, etc.*; *baseline characterisation*; *useful alongside phase-locked.*
>
> *Observation three — methodology choice has data-cost implications.* *Phase-locked produces 6× more data per measurement*; *foundation work scales accordingly*.
>
