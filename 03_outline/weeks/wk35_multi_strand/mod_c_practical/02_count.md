>
> *Six places; consistent indexing throughout*.
>
> *[pause]*
>
> *Three discipline rules for strand-index tracking.*
>
> *Rule one — index from 1 to N in canonical phase order.* *Strand 1 = canonical helix at (Δcol, Δrow) = (0, 0)*. *Strand 2 = first phase-offset (lowest non-zero Δcol)*. *Etc*. *Standard ordering avoids ambiguity*.
>
> *Rule two — colour code is permanent for the build.* *Once strand 2 is assigned 'orange', strand 2 stays orange through winding, measurement, log entries, schematic, photographs*. *No mid-build remapping*.
>
> *Rule three — log every strand operation.* *Even short interventions (touch-up of a strand's tie-off, addition of an extra heat-shrink) get logged with strand index and timestamp*. *Reproducibility hinges on this*.
>
> *[pause]*
>
> *Three secondary counts at this stage.*
>
> *Secondary count 1 — strands wound so far.* *Multi-session builds have intermediate states: '3 of 6 strands wound; 3 remaining'*. *Build-log first column reflects current state*.
>
> *Secondary count 2 — pairs of strands measured.* *6-strand has 15 pairs; 3-strand has 3 pairs*. *Mutual-inductance measurement work tracks pair-pair completion*.
>
> *Secondary count 3 — anomalies per strand.* *Build-log anomaly entries indexed by strand*. *'Anomalies count: strand 1 = 0; strand 2 = 1 (kink at pos 4); strand 3 = 0; strand 4 = 2 (entry-point off by 3°, retraced)'*. *Documentation honest*.
>
> *[pause]*
>
> *Three engineering reasons strand-index discipline matters.*
>
> *Reason one — measurements are strand-specific.* *DC resistance differs per-strand by ~5%; mutual inductances differ by phase-shift; if strand identity is unclear, measurements are uninterpretable*. *Identity preservation is identifiability*.
>
> *Reason two — replication needs the index map.* *A second practitioner replicating this build needs to know which strand was which*; *the index-and-colour map is the replication instruction*. *Reproducibility hinges on documented identity*.
>
> *Reason three — debugging requires the index.* *If a coil's behaviour deviates from prediction, the cause may be in a single strand*; *isolating the cause requires per-strand identifiable measurements*. *Diagnostic accessibility*.
>
> *[pause]*
>
