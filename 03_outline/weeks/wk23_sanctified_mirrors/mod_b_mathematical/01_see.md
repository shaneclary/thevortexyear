>
> *Projection — mainstream definition.*
>
> A *projection* π: X → Y is a map from one space (X, often higher-dim) to another (Y, often lower-dim). Projections typically *forget* some information about X to fit into Y.
>
> Examples:
>
> *Shadow.* The shadow of a 3D object on a 2D ground is a projection. The 3D shape is reduced to a 2D outline.
>
> *Photograph.* A camera projects a 3D scene onto a 2D image. Information about depth is *partially* preserved through perspective and focus, but the image is 2D.
>
> *Projection map of a torus.* T² = S¹ × S¹ has a projection π: T² → S¹ to either factor. The projection forgets one cyclic coordinate.
>
> *[pause]*
>
> *Sanctified Mirror as projection.*
>
> Conjecturally, each Sanctified Mirror is a projection π: D → ℝ² (or some 2D target) where D is the Decoqubit-style higher-dim structure.
>
> *Specific projection map: **`[ORG-INPUT]`**.* Foundation specifications would identify which projection produces each canonical Mirror.
>
> *[pause]*
>
> *Three families of projections to consider.*
>
> *Family 1 — coordinate projections.* If D = T^n, projecting onto specific coordinate axes gives 2D images. *Choose 2 of n axes; project onto them; visualise the image as 2D.*
>
> Number of distinct coordinate projections: C(n, 2) = n(n-1)/2. For n = 10: 45 distinct coordinate-axis pairs. *Up to ~45 distinct coordinate-projection mirrors.*
>
> *Family 2 — symmetry-axis projections.* Project along specific symmetry axes of D. *Each axis gives a different image.*
>
> *Family 3 — generic projections.* Project along arbitrary directions. *Many possible; specific choices produce specific images.*
>
> The canonical Sanctified Mirror set is *some specific selection* from these families. **`[ORG-INPUT]`** specifies which.
>
> *[pause]*

> *Information loss in projection.*
>
> Every projection *loses information* — that's what 'projecting' means. Specifically:
>
> *Forgotten information.* The coordinates that aren't projected onto. *Information about those coordinates is lost.*
>
> *Preserved information.* The coordinates that are projected. *Information about those coordinates is preserved.*
>
> *Holographic-style preservation.* If the projection is *cleverly chosen*, it can preserve more global information than naive coordinate projection. *This is the holographic-encoding question from Wk 22.*

> *Observation two.* The Sanctified Mirrors are *one* selection of projections; many others exist. *Marko's framework picks specific projections; these are canonical for VBM but not unique mathematically.*
