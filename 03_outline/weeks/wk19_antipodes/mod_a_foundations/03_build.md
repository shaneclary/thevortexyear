>

> Three colours of thin yarn or thread (red, blue, green work well).

> A long thin needle or wire to push thread through tight spaces.
>
> Tape or small adhesive dots to anchor thread ends.
>
> *[pause]*
>
> *Step 1 — choose colours.*
>
> *Red* for (1, 8) pair.
>
> *Blue* for (2, 7) pair.
>
> *Green* for (4, 5) pair.

> *Step 3 — mark a (2, 7) thread.* Find cell (0, 1) — value 2. Find cell (12, 4) — value 7. Push blue thread through both.

> *Step 4 — mark a (4, 5) thread.* Find cell (0, 2) — value 4. Find cell (12, 5) — value 5. Push green thread through both.

> *[pause]*
