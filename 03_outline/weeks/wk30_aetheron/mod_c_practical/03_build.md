> *Component 3 — visualisation software.* *Custom code (Python with matplotlib; LabVIEW; equivalent) that reads sensor data and animates a pattern*. *Open-source options exist.*

> *Component 4 — display.* *Computer monitor or projector showing animated pattern in real time*.

> *Component 5 — recording capability.* *Save measurement data and visualisations for later review and sharing*.
