## 4. Body (1:00–4:30)

> "The empty grid you scaffolded last week now fills.
>
> The full Rodin Number Map: six rows, twenty-four columns, 144 cells. Each cell holds a value computed from its row and column indices. The values colour the grid in three families.
>
> *[pause]*
>
> *Rows.* Six rows, indexed r = 0 to 5. Each row corresponds to a doubling power: row r holds 2ʳ mod 9.
>
> Row 0: 1.
>
> Row 1: 2.
>
> Row 2: 4.
>
> Row 3: 8.
>
> Row 4: 7 (since 16 mod 9 = 7).
>
> Row 5: 5 (since 32 mod 9 = 5).
>
> Each cell of row r holds *the same value* as the row label, *for now*. Today's grid is filled with the simple 2ʳ values; Wk 16 will complicate the picture by overlaying Fibonacci.
>
> *[pause]*
>
> *Columns.* Twenty-four columns, indexed c = 0 to 23. Each column corresponds to a power of 10. Column c holds the multiplier 10ᶜ mod 9. Since 10 ≡ 1 mod 9, every column has the same multiplier value: 1.
>
> Cell value formula: (2ʳ × 10ᶜ) mod 9 = (2ʳ × 1) mod 9 = 2ʳ mod 9.
>
> *Each row holds 24 copies of the same value.* The grid is *visually horizontal-striped* with the same value running across each row.
>
> *[pause]*
>
> *Colours.* The values 1, 2, 4, 5, 7, 8 distribute among the three family groups from Wk 7.
>
> Family 1: {1, 4, 7}. Rows 0, 2, 4 hold values 1, 4, 7. *Three rows in family-1 colour.*
>
> Family 2: {2, 5, 8}. Rows 1, 3, 5 hold values 2, 8, 5. *Three rows in family-2 colour.*
>
> Family 3: {3, 6, 9}. *No rows.* Family 3 doesn't appear in the doubling-only grid; it's the family that *doubling skips*.
>
> The grid is therefore *two-coloured horizontally* — alternating family-1 and family-2 rows — with family-3 absent.
>
> *[pause]*
>
> *What you see.* A 6×24 grid with horizontal stripes. Even rows (0, 2, 4) one colour. Odd rows (1, 3, 5) another colour. *The 3-6-9 family is missing from this grid.*
>
> *[pause]*
>
> Three observations.
>
> *Observation one.* The grid is *not yet rich*. Today's filling produces a striped pattern that doesn't reveal much beyond what Wk 4 already showed (the doubling cycle). The grid's *richness* comes when Wk 16 wraps it into a torus and Wk 17 introduces the Fibonacci structure.
>
> *Observation two.* The grid is *complete in itself for the doubling alone*. Every doubling residue appears on the grid. Nothing of the (ℤ/9ℤ)* group is missing.
>
> *Observation three.* The horizontal stripes are *the visual signature of scale invariance*. Because 10ᶜ ≡ 1 always, multiplying by 10ᶜ doesn't change the residue, so each row repeats. *The horizontal repetition is a visual proof of the scale-invariance property* you derived in Wk 9.
>
> *[pause]*
>
> Hold the grid lightly today. The stripes are correct but not exciting. Wk 16's wrap and Wk 17's Fibonacci will make the grid the central object of Q2."
