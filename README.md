# Rodin / Powell Course Project

Working directory for building a comprehensive year-long course on Marko Rodin's Vortex-Based Mathematics and Randy Powell's teachings, to be delivered to the Rodin/Powell organisation.

## Current shape (v2)

**52 weeks × 3 modules × 9 lessons × 6 min = 1,404 lessons / ~140 hours of video**

- One **theme** per week (e.g., "The Doubling Circuit", "The ABHA Torus")
- Three **modules** per week, each through a different lens (Foundations / Mathematical / Practical)
- Nine **lessons** per module, each through a fixed angle (See / Count / Build / Hear from Rodin / Feel / Apply / Compare / Defend / Connect)
- Five concepts spiral back at greater depth in Q3/Q4
